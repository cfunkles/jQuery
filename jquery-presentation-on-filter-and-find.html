jQuery .find() and .filter() Methods

.find() and .filter are traversing methods. These methods move through elements in relationship to other elements. They can find decendents of an element selected.

Example of the .find() method.

It turns only the decendants of the class selector item-ii red, the the element that is selected itself. The code here looks like this:
$( "li.item-ii" ).find( "li" ).css( "background-color", "red" );

Example of the .filter() method.

The .filter() method returns elements you specifed and not the ones you didn't specifically mention. The examples I seen like to use even and odd a lot. Here is what the code looks like:
$( "ol li" ).filter( ":even" ).css( "background-color", "aqua" );
